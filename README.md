# Draw-and-Guess
# 你画我猜简单实现

由于时间有限 只对该项目的大部分功能做简单实现
但技术栈全面

在代码中我也做了很多注释以便代码的理解

## 技术架构
vuex websocket socket
## 启动服务器

`draw-server` 服务器目录

```bash
- 安装依赖
  yarn
  
- 启动接口
  yarn start
```
服务器只用启动一次


## 启动项目

`draw-and-guess` 

```
- 安装依赖
  yarn
  
- 启动项目
  yarn serve
```
项目可以使用多个终端同时启动做测试,模拟不同的玩家
